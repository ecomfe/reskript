## 命令

使用`skr play`可以打开一个单组件的互动试调试环境。

## 参数

```
--cwd [value]           指定当前工作目录，默认为当前目录
--build-target [value]   指定调试的特性名称，默认为dev
-h, --help              显示帮助信息
```

参考[配置 - 单组件调试配置](../settings/play)了解更多配置内容。更多单组件调试的高级手段可参考[高阶应用 - 调试单个组件](../advanced/debug-component)章节的详细说明。
